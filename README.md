# Canadian Citizenship Test Practice

Practice for the Canadian citizenship exam. **[→ Open app](https://mattfung99.github.io/ca-citizenship-test/)**

Choose between a timed real test (20 questions, 45 min, pass at 15/20) or open-ended practice with configurable length and feedback. Questions are sourced from the [Richmond Public Library answer keys](https://www.yourlibrary.ca/citizenship-test-answer-keys/) and [mycitizen.ca](https://www.mycitizen.ca/full-simulation-tests.html), based on the *Discover Canada* study guide — 507 unique questions total.

Sign in to sync your history across devices. Guests can use the app without an account; results stay in the browser.

## Local dev

```bash
make        # http://localhost:8001/
make stop
```

Requires Python 3, no other dependencies.
